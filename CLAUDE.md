# AI User Mastery — Project Context

## 这个项目是什么
这是一个学习如何成为优秀 AI User 的个人学习项目。
包含学习笔记、练习、示例代码和模板。

## 目录结构
- `learning-plan.md` — 完整学习路线
- `01-mental-models/` — 心智模型笔记
- `02-prompt-engineering/` — 提示词工程，含示例
- `03-context-engineering/` — 上下文工程，含模板
- `04-claude-code-features/` — Claude Code 深度功能
- `05-workflow-design/` — 工作流设计

## 使用约定
- 每个模块的 `notes.md` 是核心笔记
- `examples/` 目录存放实际可用的示例
- `exercises.md` 是练习题和实验

## 目标读者
有一定 AI 使用经验，想系统提升的人。不需要编程背景。
